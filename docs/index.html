Geek Christmas Card Collection

The following are the Geek Christmas Card that I coded and used during the last years.

2018 - 2019

javascript

Since the Geek Christmas Card from 2016-2017 collected high success, I created a new card with the same goal: entertain the receptor involving him in a sort of Christmas tale. The main character was Mike 'the elf', who has lost the presents he crafter for this Christmas season.

2017 - 2018

javascript

I wanted to restore the original simplicity spirit of the Geek Christmas Card. This one showed Christmas pixeled-picture in an animation that wanted to emulate the cross stitch movements. The card incorporated a set of five different patterns and presents one of them randomly.

2016 - 2017

javascript

I wanted to make the receiver of the Geek Christmas Card participate of the joy of creating this special card. I designed a minigame where we need to help Mr. Snowman to find his lost hat.

2015 - 2016

java

I wanted to create a Geek Christmas Card with a retro style so I used java to type, randomly, four different small ASCII Christmas drawings.

2014 - 2015

R

Playing with the idea of fractals, I rescue the snowflake of the first Geek Christmas Card and created an R script that draws a snowflake made of snowflakes.

2013 - 2014

python

The second Geek Christmas Card reused the original idea and it was a python script drawing the face of Santa Claus.


2012 - 2013

python

The first Geek Christmas Card was a python script drawing a snowflake.
