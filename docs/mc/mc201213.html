Geek Christmas Card for 2012-2013

The first Geek Christmas Card. I built it for the Christmas of 2012. I build it in python 2.7 (now migrated to python 3) and the package turtle. The script opened a Tk window and used turtle to draw a snowflake.

This Geek Christmas Card is still operational. Once downloaded the script from the repository, open a terminal with access to python 3, and type the following command:

python mc201213.py
